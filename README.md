# IoT-Based Equipment Health Monitoring System

## Overview

This project is an IoT-based Equipment Health Monitoring System designed for monitoring critical hospital equipment.

The system continuously monitors:

- Temperature (DHT22)
- Vibration (MPU6050)

and provides:

- Real-time monitoring
- LCD display
- Blynk dashboard
- Mobile notifications
- LED status indication
- Buzzer alerts

---

## Components Used

- ESP32 DevKit V1
- DHT22 Temperature Sensor
- MPU6050 Accelerometer & Gyroscope
- 16x2 I2C LCD
- LEDs
- Buzzer
- Blynk IoT Platform
